# meg-attitudes

This repo contains all the data and code used to produce the [NI Attitudes to Minorities](https://believethedata.org/northern-irish-attitudes-to-minorities/) article.

The full workflow for the analysis behind the article is contained in the `analysis.ipynb` notebook.

Data contained in the `nilt_data` folder was downloaded from the [NILT website](https://www.ark.ac.uk/ARK/nilt/datasets/) on 2026-06-25.

Please feel free to email with any questions at jack@believethedata.org.
